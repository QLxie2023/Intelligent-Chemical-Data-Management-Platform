# 硫酸 (Sulfuric Acid)

**化学式**: H₂SO₄  
**相对分子质量**: 98.08

## 基本信息
- **组成**: [[Hydrogen|氢]] (H)、[[Sulfur|硫]] (S) 和 [[Oxygen|氧]] (O)
- **价态**: S 为 +6
- **强酸性**: Ka₁ > 0, Ka₂ = 1.2 × 10⁻²

## 物理性质
- 无色粘稠液体
- 沸点: 337°C
- 与水混溶
- 极强吸湿性

## 化学性质
- 强酸
- 强脱水剂
- 强氧化剂（浓硫酸）
- 与大多数金属反应

## 生成反应
- [[Sulfur|S]] + [[Oxygen|O₂]] → SO₂
- SO₂ + 1/2 O₂ → SO₃
- SO₃ + [[Water|H₂O]] → H₂SO₄

## 用途
- 化肥生产
- 石油精炼
- 金属处理
- 电池电解质

## 相关化合物
- [[Sulfur|S]]
- [[SulfurDioxide|SO₂]]
- [[Ammonia|NH₃]]
- [[Water|H₂O]]

## 重要性
- 化学工业最重要
- 全球最高产化学品
- 指示工业发展水平
