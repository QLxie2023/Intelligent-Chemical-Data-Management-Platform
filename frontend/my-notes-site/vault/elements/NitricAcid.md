# 硝酸 (Nitric Acid)

**化学式**: HNO₃  
**相对分子质量**: 63.01

## 基本信息
- **组成**: [[Hydrogen|氢]] (H)、[[Nitrogen|氮]] (N) 和 [[Oxygen|氧]] (O)
- **价态**: N 为 +5
- **强酸性**: Ka > 0 (超强酸)

## 物理性质
- 无色液体
- 沸点: 83°C
- 易挥发
- 与水混溶

## 化学性质
- 强酸
- 强氧化剂
- 与大多数金属反应
- 浓硫酸和浓硝酸混合为"王水"

## 生成反应
- [[Ammonia|NH₃]] → [[Nitrogen|N₂]] → NOₓ → HNO₃
- 或从[[Nitrogen|N₂]]直接合成

## 用途
- 肥料生产（硝酸盐）
- 爆炸物制造
- 金属蚀刻
- 有机合成

## 相关化合物
- [[Nitrogen|N]]
- [[Ammonia|NH₃]]
- [[Water|H₂O]]
- [[Oxygen|O₂]]

## 重要性
- 工业化学重要
- 炸药基础原料
- 农业肥料来源
