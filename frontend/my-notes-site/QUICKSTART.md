# 🚀 快速开始指南

## 第一步：查看项目结构

```
my-notes-site/
├── vault/                          # Obsidian 库文件
│   ├── elements/                   # 29 个元素的 Markdown 文件
│   └── groups/                     # 6 个分类的 Markdown 文件
├── scripts/                        # Python 脚本
│   ├── parse_markdown.py           # 知识图谱生成器
│   └── server.py                   # 本地服务器
├── output/                         # 生成的数据
│   └── knowledge_graph.json        # 知识图谱 JSON 数据
├── web/                            # 网页界面
│   └── index.html                  # 交互式可视化界面
└── README.md                       # 完整文档
```

## 第二步：在 Obsidian 中打开

1. 下载安装 [Obsidian](https://obsidian.md)
2. 打开 Obsidian
3. 点击 "打开文件夹作为库"
4. 选择项目中的 `vault` 文件夹
5. 开始浏览元素文档！

## 第三步：查看交互式知识图谱

### 方式一：使用 Python 服务器（推荐）

在项目目录打开终端，运行：

```bash
# Windows PowerShell
python scripts/server.py

# 或使用 npm 脚本
npm start
```

然后在浏览器中打开：`http://localhost:8000`

### 方式二：直接打开 HTML

直接在浏览器中打开 `web/index.html` 文件，但需要生成数据文件。

## 第四步：生成或更新知识图谱

如果修改了 Markdown 文件，需要重新生成知识图谱数据：

```bash
# 生成知识图谱 JSON
python scripts/parse_markdown.py vault output/knowledge_graph.json

# 或使用 npm 脚本
npm run generate
```

## 📊 知识图谱网页功能说明

### 左侧侧边栏
- **统计信息**: 显示节点数、关系数等
- **搜索框**: 快速查找元素或化合物

- **点击**: 显示详细信息面板

### 控制按钮
- 🔄 **重置视图**: 回到初始位置
- ⏸ **暂停/继续**: 控制图谱动画
- 📸 **导出图片**: 保存为 PNG

### 图例
- 🔴 **红色节点**: 化学元素
- 🟠 **橙色节点**: 化合物
- 🔵 **青色节点**: 分类

## 💡 常见问题

### Q: 如何添加新元素？
A: 
1. 在 `vault/elements/` 中创建新的 `.md` 文件
2. 参考现有元素的格式编写内容
3. 运行 `npm run generate` 重新生成图谱

### Q: 知识图谱不显示怎么办？
A:
1. 确保已运行 `npm run generate`
2. 检查 `output/knowledge_graph.json` 是否存在
3. 检查浏览器控制台是否有错误信息

### Q: 如何修改图表的颜色？
A: 编辑 `web/index.html` 中的 `colorMap` 对象

### Q: 支持离线使用吗？
A: 支持。下载所有文件后，离线打开 HTML 文件即可（数据已包含在 JSON 中）

## 🚨 故障排除

### 问题：浏览器显示空白
**解决**: 检查浏览器控制台（F12）的错误信息

### 问题：图谱加载缓慢
**解决**: 这是正常的，首次加载会进行物理模拟计算

### 问题：节点重叠
**解决**: 点击"暂停"按钮，手动拖拽调整位置

### 问题：Python 找不到
**解决**: 确保 Python 已添加到系统 PATH
